Core 1.5.1更新日志

  • 修复Chrome和FirefoxMousewheel事件.
  • DOMEvent的滚轮属性也监听DOM3的事件./li>
  • 修复IE8 iFrame漏洞.
  • 原生事件添加postMessage方法.
  • 修复IE7/8 setter方法.
  • 标准代值当创建新的checkbox与radio元素时.
  • 修复svg元素size bug.
  • 修复border-radius中的getter行为.
  • 修复IE8不能直接设置元素内容为html元素.
  • 添加选项来触发xhr.withCredentials不用http auth.
  • 在Request类中添加PATCH和HEAD方法.
  • 修复hasClass方法,为了兼容ECMAScript 6标准.
  • 修复IE9设置input type为"email"时的Bug.
  • 修复IE解绑销毁的元素的"submit"事件时出现的Bug.
  • 更新文档.

Core 1.5更新日志

  • Swiff从Core移除,但会在More里提供此功能.
  • 浏览器user agent的很多属性已过时,Mootools将入用功能检测代替浏览器版本检测.
  • 增加对IE11的支持, Browser.ie 将来继续正常工作 ,但在兼容模式(compatibility mode)为未定义 , 点击链接pull request 查看更多信息.
  • 对单元测试进行了大修,与Travis CI 和 SauceLabsA高度集成,新的单元测试可以在本地运行.
  • Element类速度大幅提高.(在较现代的浏览器中会使用新的API)
  • Element类添加appendHTML方法.
  • Fx类添加Fx.isPaused()方法.
  • String.contains现在遵循ECMAScript 6标准.
  • getStyle()方法背后调用getComputedStyle()方法.
  • 使用Grunt构建工具(http://gruntjs.com/).
